# Red Salud

Plataforma de telemedicina y consultas médicas online.

## Tecnologías

- **Next.js 16** (App Router)
- **React 19.2**
- **TypeScript 5**
- **Supabase** (Auth, Database, Storage)
- **Tailwind CSS 4**
- **shadcn/ui**

## Características

- Sistema de autenticación multi-rol (Pacientes, Doctores, Administradores)
- Dashboard de paciente con perfil médico completo
- Sistema de verificación de médicos con SACS (Superintendencia de Salud)
- Gestión de citas médicas
- Sistema de mensajería
- Laboratorios y resultados
- Telemedicina

## Instalación

```bash
npm install
```

## Configuración

Copia `.env.example` a `.env.local` y configura las variables de entorno de Supabase.

## Desarrollo

```bash
npm run dev
```

## Producción

```bash
npm run build
npm start
```
