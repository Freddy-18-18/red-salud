# Architecture

## Overview

This project is a monorepo managed by `pnpm`. It contains multiple applications:

- `red-salud-web`: A web application.
- `@red-salud/farmacia-desktop`: A Tauri-based desktop application for pharmacy use.
- `red-salud-desktop-medico`: A Tauri-based desktop application for medical use.

## Stack

- **Frontend**: React / Next.js
- **Desktop**: Tauri (Rust + Frontend)
- **Package Manager**: pnpm
