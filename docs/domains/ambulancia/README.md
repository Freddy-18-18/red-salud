# Ambulancia Domain

The ambulance/emergency services domain manages patient transport, dispatch, and emergency triage.

## Key Features

- **Dispatch Management**: Coordinate ambulance dispatch for emergencies
- **Patient Transport**: Track patient transport from pickup to hospital arrival
- **Triage Support**: Emergency triage workflows and documentation
- **Fleet Management**: Manage ambulance fleet availability and maintenance

## Status

This domain is in early development. The role infrastructure exists with routing support.
