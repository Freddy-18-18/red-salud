# Secretaria Domain

The secretary/administrative domain handles scheduling coordination, patient management, and billing operations.

## Key Features

- **Appointment Management**: Create, update, and cancel appointments for doctors
- **Patient Registration**: Register new patients and manage their contact information
- **Document Management**: Upload, organize, and manage medical documents
- **Billing**: Handle basic billing operations
- **Messaging**: Coordinate communication between patients and providers

## Routes

- `/dashboard/secretaria` -- Secretary management area

## Key Files

- Web dashboard: `apps/web/app/dashboard/secretaria/`
