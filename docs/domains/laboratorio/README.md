# Laboratorio Domain

The laboratory domain handles diagnostic test ordering, sample processing, and result delivery.

## Key Features

- **Test Orders**: Receive and process test orders from healthcare providers
- **Sample Tracking**: Track samples through processing stages
- **Result Delivery**: Deliver results to doctors and patients with proper access controls
- **Quality Control**: Lab quality assurance workflows

## Status

This domain is in early development. The role infrastructure exists with routing support.
