# Paciente Domain

The patient domain provides a portal for patients to manage their health journey.

## Key Features

- **Health Records**: View personal clinical history, diagnoses, and treatments
- **Appointments**: Schedule, view, and manage appointments with healthcare providers
- **Prescriptions**: Track prescription status from creation to dispensing
- **Messaging**: Communicate with healthcare providers
- **Document Access**: View and download medical documents and lab results

## Routes

- `/dashboard/paciente` -- Main patient portal
